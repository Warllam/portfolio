# Portfolio - PA / Warllam

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen.svg)](https://warllam.github.io/portfolio/)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

> A hyper-dynamic, minimalist portfolio showcasing my work as a developer and tech enthusiast.

## ✨ Features

- **Hyper Dynamic Animations** - Powered by GSAP with smooth scroll triggers and reveal effects
- **Custom Cursor** - Interactive cursor that follows mouse movement with smooth easing
- **Particle System** - Canvas-based animated particle network background
- **3D Tilt Effects** - Project cards with interactive 3D perspective transforms
- **Glitch Text Effect** - Eye-catching hero title with subtle glitch animation
- **Typewriter Effect** - Rotating taglines with typewriter animation
- **Smooth Scroll** - Buttery smooth scrolling with parallax effects
- **Scroll Progress** - Visual indicator showing page scroll progress
- **Mobile Responsive** - Mobile-first design that works beautifully on all devices
- **Dark Theme** - Sleek, minimalist dark color palette
- **Easter Egg** - Try the Konami Code 👀

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern CSS with custom properties, flexbox, and grid
- **Vanilla JavaScript** - No frameworks, pure JS
- **GSAP** - Industry-standard animation library
- **Canvas API** - For particle system rendering

## 🚀 Quick Start

1. Clone the repository:
```bash
git clone https://github.com/Warllam/portfolio.git
cd portfolio
```

2. Open `index.html` in your browser:
```bash
# Linux/Mac
open index.html

# Or use a local server
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## 📁 Project Structure

```
portfolio/
├── index.html          # Main HTML file
├── style.css           # All styles and animations
├── script.js           # JavaScript logic and interactions
└── README.md          # This file
```

## 🎨 Customization

### Colors

Edit CSS variables in `style.css`:

```css
:root {
    --bg-primary: #0a0a0a;
    --bg-secondary: #111111;
    --accent: #00d9ff;
    /* ... */
}
```

### Content

Update the content directly in `index.html`:
- Hero section: Name, title, and taglines
- About section: Bio and stats
- Skills section: Technologies and tools
- Projects section: Your work and achievements
- Contact section: Links and information

### Animations

Adjust animation parameters in `script.js`:
- Typewriter speed
- Particle count
- Scroll reveal delays
- Cursor smoothness
